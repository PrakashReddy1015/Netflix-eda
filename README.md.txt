# Netflix Exploratory Data Analysis (EDA)

## Objective
To perform exploratory data analysis on the Netflix Movies and TV Shows dataset using Python.

## Dataset
- Netflix Movies and TV Shows dataset (CSV)

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data understanding and cleaning
- Handling missing values
- Time-based analysis of content additions
- Country-wise content distribution
- Content rating distribution

## Key Insights
- Netflix content additions peaked around 2019.
- The United States is the largest content contributor, followed by India.
- Netflix focuses heavily on mature and teenage audiences (TV-MA, TV-14).
